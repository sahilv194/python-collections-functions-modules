# 1. Collections in Python

Collections are data structures used to store multiple values in a single variable.  
Python provides four built-in collection data types:

- List
- Tuple
- Set
- Dictionary

---

## 1.1 List

A List is an ordered and mutable (changeable) collection.  
It allows duplicate values.

### Features:
- Ordered
- Allows duplicates
- Changeable (mutable)
- Indexed

numbers = [1, 2, 3, 4]
print(numbers)
